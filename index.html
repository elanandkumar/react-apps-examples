App Name: RoboFriends

Description:

A very small app to show the basic usage of react components along with making an api call using fetch.

This react app is using redux for state management. Thunk is being used as middleware for asyn calls.

Any one who is new to React can refer to this application, the structure and how we can write components, styles and develop an application using react's create-react-app
